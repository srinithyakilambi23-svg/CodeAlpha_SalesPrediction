# Sales Prediction Using Python

## Overview
This project uses Machine Learning to predict sales based on advertising and marketing-related features.

The project analyzes how different advertising platforms and promotional spending affect product sales and helps businesses make better marketing decisions.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Algorithm
- Linear Regression

---

## Features of the Project
- Data preprocessing
- Data cleaning
- Exploratory Data Analysis (EDA)
- Data visualization
- Model training and testing
- Sales prediction
- Accuracy evaluation

---

## Dataset
The dataset contains information related to:
- TV advertising budget
- Radio advertising budget
- Newspaper advertising budget
- Sales values

---

## Project Workflow
1. Import dataset
2. Clean and preprocess data
3. Analyze dataset using graphs
4. Split training and testing data
5. Train Linear Regression model
6. Predict sales values
7. Evaluate model performance
8. Visualize prediction results

---

## Output
The project displays:
- Dataset summary
- Correlation analysis
- Graph visualizations
- Predicted sales values
- Model accuracy metrics

---

## Business Use Case
This project helps businesses:
- Optimize advertising budgets
- Understand customer trends
- Improve marketing strategies
- Increase sales performance
